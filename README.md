# 📅 Rubbish Calendar Web App  
*ゴミカレンダー Web アプリ*

一个简洁的小型日历网页应用，支持切换月份、跳转今天、打印 PDF，并带有基础设置功能。  
シンプルで軽量なカレンダー Web アプリ。月移動、今日へジャンプ、PDF 印刷、基本設定に対応。

---

## ✨ 功能简介 / 機能概要
- 查看日历，快速切换前后月份  
  前月・翌月への切り替え  
- **今日按钮**一键返回当前月份  
  「今日」ボタンで現在の月に戻る  
- **打印按钮**导出当月日历为 PDF  
  「印刷」ボタンで当月を PDF 保存  
- 基础设置（主题、字体等）  
  基本設定（テーマ、フォントなど）  
- 移动端布局自动适配  
  モバイル表示に最適化  

---

## 🛠 技术栈 / 技術スタック
- **HTML / CSS / JavaScript（Vanilla）**  
- 无框架，加载快速  
  フレームワーク不使用、軽量で高速  

---

## 📂 文件说明 / ファイル構成
- `index.html` — 页面结构 / ページ構造  
- `styles.css` — 样式文件 / スタイル  
- `app.js` — 日历逻辑与事件处理 / カレンダー処理  
- `manifest.json` — PWA 设置 / PWA 設定  
- `icon.png` — 应用图标 / アプリアイコン  

---

## 🚀 使用方法 / 使い方
直接打开 `index.html` 即可在浏览器中使用。  
ブラウザで `index.html` を開くだけで利用できます。

适合作为个人日历、排班、计划表，也可自行扩展。  
個人カレンダー、シフト管理、スケジュール帳などに最適で、カスタマイズも簡単です。
